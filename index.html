차돈이 뭔가요?

차돈은 자동차 정비소 사장님을 위한 매출/매입 관리 프로그램입니다. 정비일지(RO) 작성부터 고객·부품·재고 관리, 매출통계, 일일마감·결산, 계산서 인쇄, 로컬+클라우드 백업까지 정비소 운영에 필요한 장부 업무를 프로그램 하나로 처리합니다.

다른 정비 프로그램에서 내보낸 자료(CSV/엑셀)를 그대로 가져와 이어서 쓸 수 있고, 모든 자료는 내 PC의 파일 하나에 저장됩니다.

주요 기능

📋 정비일지

정비 내역(RO) 등록/조회 — 항목별 부품값·공임 분리 입력, 부품 재고 자동 차감, 완결처리/완결취소, 견적·예약, 찾기/거르기, 계산서 인쇄, 엑셀저장.

👥 고객등록

이름 · 전화번호 · 차량번호 · 년식 · 차량종류 · 관리기록을 한 화면에서 관리.

🔩 매입/부품 · 부품재고

매입일지와 부품 기준정보 통합 관리 — 외상/지불예정/지불완료 상태 관리, 총 재고 확인과 재고 부족 표시.

🪄 마법사

3단계 빠른 정비등록, 타 앱 자료 가져오기(정비일지/고객/부품), 환경설정.

📊 매출통계 · 일일마감

기간별 순이익·총매출·총매입·총외상 그래프, 하루 마감 요약과 결산 보고서 인쇄.

☁️ 백업

로컬 자동 백업 + 클라우드 백업/복원 — PC 교체나 자료 유실 시 그대로 되돌리기.

🖨️ 인쇄

정비명세서 · 견적서 · 일일마감표 · 결산 보고서를 브라우저 인쇄(Ctrl+P)로 깔끔하게.

🔄 자동 업데이트

새 버전이 나오면 앱이 스스로 알려주고, 클릭 한 번으로 내려받아 교체·재시작합니다.

사용 방법 설치부터 첫 정비 등록까지

다운로드 후 실행

위의 [차돈 다운로드] 버튼으로 차돈.exe 를 받아 원하는 폴더에 두고 더블클릭하면 끝입니다. 별도 설치 과정이 없습니다. (자료는 exe 옆에 만들어지는 data 폴더에 저장되므로, USB가 아닌 고정된 폴더에 두는 것을 권장합니다.)

인증키 입력

첫 실행 시 인증키(CHDN-XXXX-XXXX-XXXX) 입력 창이 뜹니다. 구매/문의 시 받은 키를 입력하면 해당 PC에 자동으로 등록됩니다. 인증키가 없다면 mokky@mokky.store 로 문의하세요.

환경설정 (선택)

마법사 → 환경설정에서 상호·전화·주소를 입력해 두면 계산서 등 인쇄물 상단에 표시됩니다. 계산서 기본 방식과 자동 백업도 여기서 설정합니다.

기존 자료 가져오기 (선택)

다른 정비 프로그램을 쓰고 있었다면, 그 프로그램의 엑셀(CSV) 내보내기로 저장한 파일을 마법사 → 타 앱 자료 가져오기에서 반입하세요. 정비일지·고객·부품을 지원하며 열 이름을 자동으로 인식합니다.

정비 등록 시작

정비일지 → 신규 또는 마법사의 3단계 빠른 등록으로 정비 내역을 기록하세요. 완결처리하면 계산금액이 채워지고, 일일마감에서 하루를 마감하면 자동 백업까지 됩니다.

자주 묻는 질문

실행할 때 "Windows의 PC 보호" (SmartScreen) 경고가 떠요.

코드 서명이 없는 프로그램이라 처음 실행 시 파란 경고 창이 뜰 수 있습니다. 추가 정보 → 실행을 누르면 정상 실행됩니다. 이 사이트에서 받은 파일만 실행하세요.

업데이트는 어떻게 하나요?

따로 할 일이 없습니다. 차돈은 시작할 때 이 사이트의 버전 정보를 확인해서 새 버전이 있으면 "새 버전이 나왔습니다" 안내를 띄웁니다. [예]를 누르면 앱이 스스로 내려받아 교체하고 다시 시작합니다. 메뉴 도움말 → 업데이트 확인으로 직접 확인할 수도 있습니다.

내 자료는 어디에 저장되나요?

실행 파일 옆의 data\chadon.db 파일 한 개에 모두 저장됩니다. 이 파일만 복사해 두면 그대로 백업이 됩니다. 프로그램 안의 [백업] 메뉴로 로컬+클라우드 백업을 쓰면 더 안전합니다.

PC를 바꿨어요. 자료를 옮기려면?

새 PC에서 차돈을 받아 실행하고 인증키를 등록한 뒤(기기 교체는 관리자에게 문의), [백업] → [클라우드에서 복원]을 누르면 마지막 클라우드 백업이 그대로 복원됩니다. 또는 이전 PC의 data\chadon.db 파일을 새 PC의 exe 옆 data 폴더로 복사해도 됩니다.

어떤 PC에서 쓸 수 있나요?

Windows 10 이상 64비트 PC를 지원합니다. 인터넷은 인증·업데이트·클라우드 백업에만 사용되며, 잠시 끊겨도 7일간은 정상 동작합니다.
